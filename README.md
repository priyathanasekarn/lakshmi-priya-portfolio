## üåê Hosting on GitHub Pages

1.  **Initialize Git**:
    Open your terminal/command prompt in this folder and run:
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    ```

2.  **Create a Repository on GitHub**:
    - Go to [GitHub.com](https://github.com) and sign in.
    - Click the "+" icon in the top right and select "New repository".
    - Name it (e.g., `my-portfolio` or `lakshmi-priya-portfolio`).
    - **Do not** initialize with README/gitignore (you already have them).
    - Click "Create repository".

3.  **Push Code**:
    Follow the instructions shown on GitHub to push an existing repository:
    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    git branch -M main
    git push -u origin main
    ```

4.  **Enable GitHub Pages**:
    - Go to the **Settings** tab of your repository.
    - Click **Pages** in the left sidebar.
    - Under **Source**, select `main` branch and `/ (root)` folder.
    - Click **Save**.
    - Wait a minute, and you will see your live URL (e.g., `https://your-username.github.io/repo-name/`).
